############################################################
#
# DOWNLOADING LROSE
#
# This readme details which files you should download
# for building, running or installing lrose projects.
#
############################################################

Download locations
------------------

  https://www.eol.ucar.edu/software/lrose-core
  ftp://ftp.rap.ucar.edu/pub/titan/lrose

Multiple files for download
---------------------------

The LROSE distribution has been packaged into 3
separate downloadable tar files:

(a) Source distribution:

  lrose-yyyymmdd.src.tgz

(b) 64-bit binary distribution:

  lrose-yyyymmdd.x86_64.tgz

(c) 32-bit binary distribution:

  lrose-yyyymmdd.i686.tgz

Build from source
-----------------

To build lrose from source,
download the latest source distribution:

  lrose-yyyymmdd.src.tgz

and then follow the instructions in:

  README_BUILD_LINUX.TXT - LINUX   builds
  README_BUILD_OSX.TXT   - MAX OSX builds

Use 64-bit binary distribution
------------------------------

To run using pre-compiled 64-bit binaries,
download the latest binary release:

  lrose-yyyymmdd.x86_64.tgz

and then follow the instructions in:

  README_INSTALL_BINARY.TXT

Use 32-bit binary distribution
------------------------------

To run using pre-compiled 32-bit binaries,
download the latest binary release:

  lrose-yyyymmdd.i686.tgz

and then follow the instructions in:

  README_INSTALL_BINARY.TXT
