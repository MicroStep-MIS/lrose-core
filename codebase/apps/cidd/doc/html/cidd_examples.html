CIDD Examples Page

If you have CIDD in your path and your browser set properly, then you can click on a link, like those below, and CIDD will start up using that parameter file. Otherwise, use the syntax:
'CIDD -p http://www.rap.ucar.edu/CIDD/configuration_name.CIDD'
from the command line

Available Configurations:

  • Taiwan AOAWS Data
  • Live Colorado Front Range Radars
  • Icing and Turbulence Data
  • National Covection Display
  • OCND Data - Pacific Ocean
  • Redstone Data - Dual Doppler Data set
  • South Africa Data
  • Sterling Va. Nowcast Data
  • Sydney Australia Nowcast Data
  • Navy TEP Data

How to Get Netscape to open CIDD:

  • Edit Netscape Preferences->Navigator->Applications
  • Select New...
  • Enter 'CIDD Parameter Files' in the Description box
  • Enter 'application/cidd' in the MIMEType box
  • Enter 'CIDD' in the Suffixes Box
  • Select Application check box. Enter '/rap/bin/CIDD -p %s'
  • Click OK-> OK
  • Click on the Links above to Start the Display
