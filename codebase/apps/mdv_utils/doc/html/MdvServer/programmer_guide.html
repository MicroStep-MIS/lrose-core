MdvServer Programmer's Guide

This document gives information useful to a programmmer maintaining the MdvServer code and describes the how programmer's should write clients of the MdvServer.

The MdvServer serves MDV data to requesting clients using the MDV protocol. The MDV protocol is described in the mdv/mdv_client.h include file.

MdvServer Clients

Clients should use the MDV_put and MDV_get routines in the MDV library (described in mdv/mdv_client.h) to retrieve MDV data using the RAP URL. These routines will store/retrieve data from the proper data set as described by the URL. If the data set is local, the routines will access the data directly from the disk. Otherwise, the routines will communicate with the proper server, using the server manager of the DIDSS system, to retrieve the data for the client.
