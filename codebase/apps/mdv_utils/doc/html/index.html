MDV Utility Applications

The applications in the mdv_utils directory are all generic MDV utilities. The current MDV utilities are:
Mdv2Vil

MdvMerge

MdvServer
The MdvServer serves MDV data to requesting clients using the MDV protocol. The MDV protocol is described in the mdv/mdv_client.h include file.
